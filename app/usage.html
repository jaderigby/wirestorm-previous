Layout

+panel()

Description: Creates a panel content wrapper including substantial whitespace above and below, as well as centering content while allowing background color to extend to the edge of the browser. Panels should be stacked one upon another and not nested.

  • arg1 = [ string ]
    – recieves 'light', 'dark', or can be left blank for a clear background.

Code


+panel()
	h3 Demo
	p
		+blockletter(30)

Dynamic Elements

+modal(arg1, arg2)

Description: Creates a button linking to a modal window. Content of modal is anything indented underneath the modal element (see below). Content of the modal can be any html (content, images, etc)

  • arg1
    [ string ] – button text
  • arg2
    [ string (optional) ] – position of button ("left", "center", or "right")

Code


+modal("Learn More")
    p Content of the modal     
 
// or . . . 
 
+modal("Learn More", "center")
    p Content of the modal including an image
   +image(100)

+tabs(arg1)

Description: Creates a section of tabs. The +tabs() element is used in conjunction with the +tab() element (singular). Each tab can be defined under a +tab() element. Refer to the code example below.

  • arg1
    [ array ] – title of each tab, as a list of strings.

Code


+tabs(['Tab 1', 'Tab 2', 'Tab 3'])
    +tab()
        p This content will be contained within the first tab.
    +tab()
        p This is the content for the second tab.
    +tab()
        p And finally, the content for the third tab.
  • Tab 1
  • Tab 2
  • Tab 3

Loops

+loop(arg1)

Description: Makes a copy of the content for the number of times passed in as "arg1".

  • arg1
    [ integer ] – number of times to loop.

Code


+panel('light')
	- var i = 0
	+loop(3)
		article.border-wrap
			h3= titles[i++] + ' Title'
			p
				+blockletter(35)

+column-loop(arg1, arg2)

Description: Makes a copy of the content for the number of times passed in as "arg1", as columns. The copied column must be contained within an "article" tag.

  • arg1
    [ integer ] – number of columns to create.
  • arg2
    [ string ] – pass in the string "equalize", if you would like each column to be equal in height. Then, add "data-equalize" to each wrapper that acts as the column's height (See code below.)

Code


+panel('dark')
	+column-loop(4,'equalize')
		article
			div.wrap(data-equal-height)
				p
					+porthole(150, 'center')
				p
					+blockletter(15)

Media

+image(arg1, arg2, arg3)

Description: Creates a dummy image element. You can leave all parameters off, and allow the image to take on the dimensions of the parent, as well.

  • arg1
    [ integer, string ] – width of element. Note: you can pass in a string for percentages, such as "100%".
  • arg2
    [ integer, string ] – height of element. Note: you can pass in a string for percentages, such as "100%".
  • arg3
    [ string ] – position of element; can pass in the strings "left", "center", or "right"

Code


+image(450, 250)
 
// or . . . 
 
article
   +image('100%', 150, 'center')
